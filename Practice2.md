Что произойдет:

- При `case 1`: выполнится `i = 1`, затем `break` завершит switch
- При `case 2`: выполнится `i = 2`, но поскольку нет `break`, выполнение "провалится" в `case 3` и выполнится `i = 3`, затем сработает `break`
- При `case 3`: выполнится `i = 3`, затем `break`

Пример выполнения:
```java
int value = 2;
int i = 0;

switch(value) {
    case 1:
        i = 1;
        break;
    case 2:
        i = 2;  // выполнится
    case 3:
        i = 3;  // тоже выполнится из-за проваливания
        break;
}

System.out.println(i); // Выведет 3, а не 2!
```
