# Ответы на вопросы:

**Практика 2:**
Если у переопределенного метода не совпадает тип возвращаемого значения - будет ошибка компиляции.

**Практика 3:**
Аннотация @Override помогает обнаружить ошибки когда:

1. Опечатка в имени метода (например makeSoud вместо makeSound)
2. Неправильные параметры метода
3. Попытка переопределить несуществующий метод
4. Несовпадение типа возвращаемого значения
