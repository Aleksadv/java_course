# Ответы на практические задания:

## Практика 1: try без catch и finally
**НЕТ, нельзя**
```java
try {
    ...
}
// ОШИБКА КОМПИЛЯЦИИ
```

## Практика 2: try только с finally
**ДА, можно**

```java
try {
    ...
} finally {
    ...
}
```

## Практика 3: try с двумя finally
**НЕТ, нельзя**

```java
try {
    ...
} finally {
    ...
} finally { // ОШИБКА КОМПИЛЯЦИИ
    ...
}
```

**Правила использования try:**
- try должен иметь хотя бы один catch ИЛИ finally
- finally может быть только один на каждый try
- finally выполняется ВСЕГДА (кроме System.exit() или сбоя JVM)
- Порядок блоков: try → catch → finally
