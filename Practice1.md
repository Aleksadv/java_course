## Результат выполнения кода

```java
int a1[] = {1,2,3,4,5};
int a2[] = {6,7,8,9,10};
a1 = a2;
```

**Что произойдет:**

1. Создается массив `a1` с элементами `[1,2,3,4,5]`
2. Создается массив `a2` с элементами `[6,7,8,9,10]`
3. При выполнении `a1 = a2`:
   - Переменная `a1` начинает ссылаться на тот же массив, что и `a2`
   - Оба массива `a1` и `a2` теперь указывают на один и тот же массив `[6,7,8,9,10]`
   - Исходный массив `[1,2,3,4,5]` теряет ссылку
