# Три двойные кавычки - Text Blocks 

## Зачем нужны:
Text Blocks (текстовые блоки) позволяют удобно работать с многострочным текстом без экранирования и конкатенации.

## Преимущества:
1. **Читаемость** - сохраняют форматирование
2. **Удобство** - не нужны escape-последовательности
3. **Поддержка форматирования** - JSON, HTML, SQL и т.д.

## Примеры сравнения:

### Без Text Blocks:
```java
String html = "<html>\n" +
              "    <body>\n" +
              "        <h1>Title</h1>\n" +
              "    </body>\n" +
              "</html>";
```              
**С Text Blocks:**
```java
String html = """
    <html>
        <body>
            <h1>Title</h1>
        </body>
    </html>
    """;
```    

**Особенности:**
- Начинаются с """ и перевод строки
- Сохраняют отступы (убирают общие ведущие пробелы)
- Автоматически добавляют переводы строк
- Можно использовать внутри выражений
